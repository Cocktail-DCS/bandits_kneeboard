SITUACIÓN

Tu tarea es realizar una patruya aerea sobre la zona objetivo evitando que se acerquen aviones enemigos al grupo de STRIKE

IMPORTANTE

Configrar el bullseye para realizar un correcto reporte de enemigos

Anunciar lanzamientos de misiles

Ruta

Salimos desde el punto de espera para cumplir con el TOT barriendo el horizonte monitorizando cualquier contacto que pueda estar por la zona

Este recorrido tambien es perfecto para sincronizarse con el resto de vuelos de CAP

Procedimiento en orbita

Una vez establecidos en orbita, organizarse con los distintos vuelos de CAP para cubrir todo el espacio aereo, una distribucion buena para 2 vuelos sería que durante la orbita mientras unos monitorizan el este, los otros el oeste

También usar un volumen de escaneo del radar correcto ayuda a detectar amenazas con mayor facildad ya que centramos el escaneo el el area importante
